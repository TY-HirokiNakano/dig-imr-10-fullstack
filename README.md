# マラソン練習アプリ

## 概要

このアプリは、マラソンの目標タイムを入力することでジョギングでの推奨ペース、目標のレベルを表示します。（2026/05/28時点、ハーフマラソン向けに限定）
練習のモチベーションを高めるねらいがあります。

## 事前準備

以下がインストールされていることを確認してください。

- [ ] [Node.js](https://nodejs.org/ja/) (動作確認のバージョン24.14.1)
- [ ] [PostgreSQL](https://www.postgresql.org/) (動作確認のバージョン18.3)

## セットアップ手順

依存関係のインストール

```sh
npm install
```

DBのセットアップ

```sh
npm run db:migrate
npm run db:seed
```

### 本番環境

フロントエンドのビルド

```sh
npm run build
```

サーバーの起動

```sh
npm start
```

アクセス先: http://localhost:3000

### 開発環境

バックエンドの起動

```sh
npm run dev
```

フロントエンドの起動

```sh
cd front
npm run dev
```

アクセス先: http://localhost:5173

## アプリの詳細

- ペース計算: 目標タイムを入力すると、ジョギングでの推奨ペースを表示
- レベルチャート: ペース計算を実行すると、目標タイムに対応するレベルを表示
- レベル設定編集: レベルのラベルやタイムの閾値を編集

## リソース

- [ダニエルズのランニング・フォーミュラ](https://www.sportsclick.jp/products/detail/11289)の考え方をDBのseedファイル作成に使用

## npmのスクリプト

| コマンド                  | 説明                                                                        |
| ------------------------- | --------------------------------------------------------------------------- |
| `npm start`               | サーバー起動                                                                |
| `npm run dev`             | サーバー起動（ファイル変更時に自動で再起動）                                |
| `npm run build`           | frontのリポジトリをビルドして、ルートディレクトリ直下のpublicフォルダに出力 |
| `npm run db:migrate`      | マイグレーションを実行                                                      |
| `npm run db:migrate-make` | マイグレーションファイル作成                                                |
| `npm run db:seed`         | シードファイルをDBに追加                                                    |
| `npm run db:seed-make`    | シードファイル作成                                                          |
| `npm run db:reset`        | DBをリセット、マイグレーション実行、シードファイル投入                      |
